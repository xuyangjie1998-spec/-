MOD制作教程

完整的新势力、新武将、新物品创建流程指南

🏰 添加新势力

步骤1: 创建势力数据
MOD制作向导 → 新增势力卡片 → 填写国号/君主/颜色/首都
步骤2: 一键联动数据
势力编辑器 → 一键联动 → 自动创建 Color.ini + City.ini + City01-10.ini
步骤3: 配置君主武将
确保君主武将的 Lord 字段 = 新势力编号
步骤4: 配置城池归属
City01-10.ini 中设置城池 Owner 为新势力编号
步骤5: 配置势力颜色
Color.ini 编辑器中调整旗帜 RGB 颜色
关联文件: Nation.ini → Color.ini → City.ini → City01-10.ini → General01.ini

⚔️ 添加新武将+头像CG

步骤1: 制作头像CG
准备128x128 PNG图片 → 头像面板 → 导入自定义头像 → 自动转为 Shape/GenFace/XXXX.shp
步骤2: 创建武将数据
MOD制作向导 → 一键创建武将 → 填写编号/姓名/武力/智力/头像ID等
步骤3: 配置初始技能
自动联动创建 DefSkill.ini 条目（武将技/军师技/特性）
步骤4: 配置出场位置
自动联动创建 General02.ini 条目（10个剧本出生地）
步骤5: 配置战场造型
OBD编辑器 → 选择 BFGen.obd → 创建造型序列 → ResID 关联
头像规格: 128×128像素, 256色调色板, 保存为 Shape/GenFace/XXXX.shp

💎 添加新物品+图标

步骤1: 制作物品图标
准备64x64 PNG图片 → 物品编辑器 → 导入图标 → 转为 Shape/ThingIcon/XXXX.shp
步骤2: 创建物品数据
MOD制作向导 → 一键创建物品 → 填写编号/名称/类型/价格/图标ID
步骤3: 填入名称和描述
自动创建 TermText 名称(14000+No) + 描述(15000+No)
步骤4: 配置武器特效
ScriptNo 字段: 0=无, 1=吸血, 3=贯穿, 6=冰冻等
步骤5: 商店上架/锻造
CitySellItem → 添加物品到城池商店 | ItemEnhance → 添加锻造配方
AI装备: Price决定AI装备优先级, Price=1则AI丢弃 | IsRare=6仅事件获取

📖 AI行为逻辑详解

基于逆向分析，解释AI各项决策的底层机制
决策类型机制说明关联参数
⚔ 出兵决策AI根据外交关系等级决定是否出兵。对玩家需要1.5倍兵力优势才出兵(Float05)；对AI只需0.75倍(Float06)。设Float06为负数可使AI之间频繁混战，武将极速升级。No.139 外交度攻打机率
🏃 撤退判断战前选部队时，AI根据战力比(Int00-01)决定是否撤退。全改为0可禁止AI撤退，改为高值则AI更容易撤退。No.134 撤退战力比
📈 武将升级未出战武将每季自动获得经验(Int00=50)。AI武将在各难度下获得额外倍率(Float00-05)。配合No.133 AI战斗经验倍率，双管齐下加速AI成长。No.137 自动经验
No.133 战斗经验倍率
🔍 搜索行为AI搜索人才/物品成功率(Int01=40)。值越大AI搜东西越快，但设太大(100)则双方都搜不出。玩家搜索成功率(Int00=10)独立于AI。No.121 搜索成功率
No.132 人才搜索速率
🏰 内政开发城市开发速度由Float00-03倍率控制。改太大会导致武将无法被捕获(需配合调整兔脱特性值)。No.119 城市内政公式
🤝 外交关系每10天自动调整外交值：外交<50时增加(Int01=1)，外交>50时减少(Int02=3)。开战后外交值按战前关系减少(No.140)。No.138 外交度增减
No.140 战后外交减少
👑 封官逻辑AI每季为每个武将选Cost最低且满足(武力、智力、等级)条件的官职。弱武将官职少→封官快→官阶高→AI优先派出战。Title.ini Cost/Str0/Int0
见官职编辑页

见物品编辑页
👻 NPC刷新大地图NPC(仙人、山贼等)出现数量由Int01控制，改上千可大幅增加。Float00=停留时间倍数(默认5)。No.238 事件部队参数

AI封官/装备机制说明

核心规则: AI每季为每个武将选一个Cost最低且满足条件的官职封官。

封官条件: 武将需满足 Str0 ≤ 武力 ≤ Str1, Int0 ≤ 智力 ≤ Int1, 等级 ≥ Level

策略建议: 弱武将的官职设高Cost → 封官慢 → 官阶低；强武将的官职设低Cost → 封官快 → 官阶高

AI装备规则: Thing.ini中 Price 决定AI是否装备。Price=1时AI会丢弃该物品; Price越高AI越重视。IsRare=6仅事件获取。

策略建议: 提高神兵Price → AI主动装备; 降低垃圾装备Price → AI不捡; 在Variable.ini的No.133调整AI战斗经验倍率。

📖 帮助文档

            
        

        

            
快速入门

            
San7ModMaker 是三国群英传7 MOD制作工具，支持编辑游戏中的各种配置文件。

            
基本操作

            
    
                
  • 加载：点击"加载数据"或"载入"按钮读取游戏文件
    • 
                
  • 编辑：在右侧详情面板修改字段值
    • 
                
  • 保存：点击"保存"按钮将修改写回游戏文件
    • 
                
  • 新增：点击"新增"按钮创建新条目
    • 
                
  • 删除：选中条目后点击"删除"按钮
    • 
                
  • 搜索：在搜索框中输入关键词过滤列表
    • 
                
  • 撤销：Ctrl+Z 撤销上一步操作
    • 
            

            
核心概念

            
    
                
  • TermText：游戏文本管理，ID 13000+为兵种名，14000+为物品名，25000+为武将名
    • 
                
  • DefSkill：武将个人特性，编号对应 General01.ini 中的武将编号
    • 
                
  • General02：武将出生地/登场剧本配置
    • 
                
  • OBD：兵种/武将千人战模型定义
    • 
                
  • Script.so：游戏脚本库，SuperStart控制必杀武器系别
    • 
            

            
数据联动

            
    
                
  • 修改武将编号 → 自动同步 DefSkill / General02 / TermText
    • 
                
  • 新增兵种 → 自动创建兵符物品
    • 
                
  • 修改城池 → 自动同步 TermText 名称
    • 
                
  • 修改必杀技 → 自动同步 TermText 名称
    • 
            

            
MOD 制作流程

            
    
                
  1. 设置游戏目录（Settings → 游戏目录）
    2. 
                
  2. 使用「MOD向导」快速创建武将/兵种/势力
    3. 
                
  3. 在对应编辑器中微调属性
    4. 
                
  4. 使用「备份还原」保存当前状态
    5. 
                
  5. 使用「数据校验」检查配置正确性
    6. 
                
  6. 使用「MOD管理」打包发布
    7.
